# Guidelines for Claude

## Git commits

- Do **not** add `Co-authored-by:` trailers to commit messages.
- Do not add "Generated with Claude Code" or similar attribution lines.
- Keep messages short. A subject line alone is usually enough. Add a body only
  when it carries information the diff does not, such as a non-obvious cause or
  a rejected alternative — not to restate or summarise the change.
